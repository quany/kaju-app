# 架构说明
这是一个轻量级开发方案，它不需要android和ios工程，expo这个App会提供各种RN环境；
## 运行环境
- 安装nodejs 6以上的
- git
- python 2.7 以上
- 安装yarn

```
npm install -g yarn
```
- 切换yarn

```bash
yarn config set registry https://registry.npm.taobao.org --global
yarn config set disturl https://npm.taobao.org/dist --global
```
- 安装 react-native-cli

```bash
yarn add global react-native-cli
```

## 工程运行
- 首先下载工程
- cd vapp,安装node包：

```bash
yarn install
```
耐心等待安装完成
- 安装完成后,运行该工程
```bash
npm start
```
